# 🌍 Language Detection using Transformers

This project detects the language of any input text using the pretrained XLM-RoBERTa Language Detection model from Hugging Face.

## 🚀 Features
- Detects 100+ languages
- Built with Streamlit
- Powered by Hugging Face Transformers
- User-friendly web interface

## 🛠️ Technologies Used
- Python
- Streamlit
- Transformers
- PyTorch

## 📂 Project Structure
```
language-detection/
│── app.py
│── requirements.txt
│── README.md
```

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
